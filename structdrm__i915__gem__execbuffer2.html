__u64 drm_i915_gem_execbuffer2::cliprects_ptr

This is a struct drm_clip_rect *cliprects if I915_EXEC_FENCE_ARRAY is not set. If I915_EXEC_FENCE_ARRAY is set, then this is a struct drm_i915_gem_exec_fence *fences.

This is a struct drm_clip_rect *cliprects if I915_EXEC_FENCE_ARRAY & I915_EXEC_USE_EXTENSIONS are not set.

If I915_EXEC_FENCE_ARRAY is set, then this is a pointer to an array of struct drm_i915_gem_exec_fence and num_cliprects is the length of the array.

If I915_EXEC_USE_EXTENSIONS is set, then this is a pointer to a single struct i915_user_extension and num_cliprects is 0.
