Detailed Description

struct prelim_drm_i915_gem_wait_user_fence

Wait on user fence. User fence can be woken up either by,

  1. GPU context indicated by 'ctx_id', or,
  2. Kerrnel driver async worker upon PRELIM_I915_UFENCE_WAIT_SOFT. 'ctx_id' is ignored when this flag is set.

Wakeup when below condition is true. (*addr & MASK) OP (VALUE & MASK)
