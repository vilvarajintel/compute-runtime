Detailed Description

struct prelim_drm_i915_gem_clos_free

Free off a previously reserved CLOS set. Any corresponding Cache Reservations that are active for the CLOS are automatically dropped and returned to the Shared set.

The clos_index indicates the CLOS set which is being released and must correspond to a CLOS index previously reserved.
